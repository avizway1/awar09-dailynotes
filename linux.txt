Secure: Fewer virus and strong permissions
Stable : Run for months/years without reboot
Fast : It consumes very less resources than windows.

Open source : we cna access the source code, we can modify, we can redistribute..

In linux everything starts from / (root --> Top level (windows c:\ in windows))

/root 	--> Home directory for roor user (In windows --> c:\users\administrator)
/home 	--> Home directory for regular users (all other users than root)
/bin 	--> Contains essential linux commands (ls, cp, pwd, cat)
/sbin	--> System admin commands (reboot, iptables, blkid, mkfs)
/etc	--> Configurations files (Settings for the servcies)
/var	--> Variable data changes freq. (logs, documentrootpath, databases)
	/var/log --> System log files
/dev	--> Device files (hard disks, usb)


/tmp	--> Temporary files (auto cleaned)
/usr	--> User programs (C:\program files)
/opt	--> Operational/thirdparty files
/proc	--> Liove system info i.e; cpu, ram
/boot	--> Boot files i.e; kernal, grub, vmlinuz
/lib	--> Shared libraries (similar to .dll files windows)
/mnt	--> Manual mount points

vim : vi Improved : Text editor for linux OS.

vi : original editor
vim : Enhanced / Modified vi editor

VIm has 2 modes..

Currently in "Command Mode"--> Press "i" to get into Insert Mode (to write the data) 
currently in "Insert Mode" --> Press "esc" to get into Command mode 

:wq			: Write the info the file and quit from the editor
:q!			: Quit without saving the changes to the file

o			: Open a new line at cursor
x			: Delete the character

dw			: Delete the word
dd			: Delete the entire line
3dd			: Delete the next 3 lines from the cursor
100dd		: Deletes 100 lines from the cursor

nG			: Jumps to the specific line number
22G			: Jump the cursor to line number 22
