.git directory contains

files:
config : repo specific confis i.e; remote URLs and branch tracking
description : used by gitweb to display desc. Not applicable for latest git.
HEAD : Point to the current checkout-out branch or specific commit hash. 
index : The staging area where git prepares the content for our next commit.

Dir: 
objects/	: directory structures, compressed file contents (blobs), commit history and tags.
refs/		: Stores pointers to commits.
hooks/		: cotnains customizable scripts that git runs automatically before or after commit / push.
info/		: addl global files
logs/		: keeps the history of changes made to references.
